Front-end pattern libraries
What is a pattern library?
A modular collection of all the elements in your product’s user interface, together with code snippets for developers to copy and paste as needed to implement those elements.
It includes common UI components like buttons, form-input elements, navigation menus, modal overlays, and icons.
